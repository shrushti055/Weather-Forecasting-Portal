# WeatherPortal

This Weather Portal is a responsive weather application built with HTML, CSS, and JavaScript for the frontend, and Java (Spring Boot) for the backend. It provides hourly and weekly forecasts, real-time weather updates, and an automatically updating map view for any location in the world.
